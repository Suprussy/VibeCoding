사용자는 `VibeCoding` 디렉터리 구조에서 Study 단위로 프로젝트를 진행하는 개발자/빌더.

- 단일 HTML 파일(순수 프론트엔드) 방식의 빠른 프로토타이핑 선호
- 외부 라이브러리는 CDN 방식으로 로드 (번들러 없음)
- AI 기능 연동(OpenRouter, API 키 직접 입력 방식) 경험 있음
- PRD는 구현 팀이 바로 코딩에 착수할 수 있는 수준의 구체성 요구
- 한국어 응답 선호
