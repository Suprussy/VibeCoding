학생 성적 현황표를 생성합니다. 특정 학생 이름이 주어지면 해당 학생만, 없으면 전체 학생을 조회합니다.

인자: $ARGUMENTS (선택 — 학생 닉네임. 비어있으면 전체 조회)

---

## 실행 절차

### 1단계: 데이터 로드

`quiz-game/teacher-data/results.json` 파일을 읽습니다.
파일이 없으면 `[오류] results.json 파일이 존재하지 않습니다. 먼저 학생 성적 데이터를 준비해주세요.` 출력 후 중단합니다.

### 2단계: 학생 필터링

- 인자가 있으면: 해당 닉네임의 기록만 필터링
- 인자가 없으면: 전체 학생 대상

닉네임이 존재하지 않으면 `[오류] '{닉네임}' 학생의 기록을 찾을 수 없습니다.` 출력 후 중단합니다.

### 3단계: 성적 현황표 출력

#### 전체 조회 시

학생별로 각 카테고리의 **최고 점수(best score)** 를 매트릭스 형태로 출력합니다.
같은 학생이 같은 카테고리를 여러 번 응시했으면 최고 점수만 표시하고, 응시 횟수도 함께 표기합니다.
미응시 카테고리는 `-`로 표시합니다.

출력 형식:
```
## 전체 학생 성적 현황

| 학생   | 성경 | 힙합 | NBA | MLB | 평균  | 응시 수 |
|--------|------|------|-----|-----|-------|---------|
| {이름} | N점  | N점  | N점 | N점 | N.N점 | N회     |
| ...    |      |      |     |     |       |         |

총 학생 수: N명 / 총 응시 기록: N건
```

정렬 기준: 평균 점수 내림차순

#### 개별 학생 조회 시

해당 학생의 모든 응시 기록을 시간순으로 보여줍니다.
재응시한 카테고리가 있으면 점수 변화(향상/하락)를 표시합니다.

출력 형식:
```
## {닉네임} 학생 성적 상세

### 카테고리별 최고 기록
| 카테고리 | 최고 점수 | 최고 기록 소요시간 | 응시 횟수 |
|----------|-----------|-------------------|-----------|
| {카테고리} | N/10    | N초               | N회       |

### 전체 응시 이력 (시간순)
| 날짜       | 카테고리 | 점수 | 소요시간 | 변화    |
|------------|----------|------|----------|---------|
| 2026-06-10 | 성경     | 8/10 | 142초    | (첫시도) |
| 2026-06-14 | 성경     | 9/10 | 125초    | +1점    |

### 종합 평가
- 평균 점수: N.N점
- 가장 잘한 카테고리: {카테고리} (N점)
- 가장 약한 카테고리: {카테고리} (N점)
- 총 응시 시간: N초
```
