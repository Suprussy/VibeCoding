## 실행 절차

### 1단계: 데이터 로드

`quiz-game/teacher-data/results.json` 파일을 읽습니다.

### 2단계: 순위 산출

각 학생의 **카테고리별 최고 점수(best score)** 를 기준으로 순위를 매깁니다.
동점일 경우 소요시간이 짧은 학생이 상위입니다.

### 3단계: 출력

#### `all`이거나 인자가 없는 경우

**종합 순위**와 **카테고리별 순위**를 모두 출력합니다.

종합 점수 = 응시한 카테고리들의 최고 점수 합산 (미응시 카테고리는 0점 처리)

순위 표시 규칙:
- 1위: 🥇
- 2위: 🥈
- 3위: 🥉
- 미응시 학생은 해당 카테고리 순위에서 제외

#### 특정 카테고리 지정 시

해당 카테고리 순위만 출력합니다. 추가로 해당 카테고리의 통계도 표시합니다:
- 응시자 수
- 평균 점수
- 최고/최저 점수
- 평균 소요시간
