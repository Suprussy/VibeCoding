지정된 범위의 퀴즈 문제를 검토하여 난이도와 정답 분포를 분석합니다.

$1번부터 $2번까지의 문제를 검토합니다.
문제 번호는 각 카테고리 파일 내 배열 인덱스 기준(1부터 시작)이며,
`quiz-game/src/data/` 디렉터리의 bible.ts, hiphop.ts, nba.ts, mlb.ts 전체에서 해당 범위의 문제를 대상으로 합니다.

예: $1=3, $2=7이면 각 카테고리 파일의 3번째~7번째 문제를 검토합니다.

## 검토 항목

### 1. 난이도 평가
각 문제를 아래 기준으로 난이도를 분류하세요.

- **상 (Hard)**: 구체적 수치, 통계, 연도 등 암기 없이는 풀 수 없는 문제. 정답과 오답 선택지 간 구분이 어려움.
- **중 (Medium)**: 해당 분야에 관심 있는 사람이라면 추론 가능하지만, 정확한 지식이 필요한 문제.
- **하 (Easy)**: 일반 상식 수준으로 풀 수 있는 문제. 마니아 난이도에 부적합.

CLAUDE.md의 기준에 따라 마니아/전문가 수준이어야 하므로, **하(Easy)** 문제가 있으면 난이도 상향을 권고하세요.

### 2. 정답 분포 분석
해당 범위 내 문제들의 `answer` 값(0, 1, 2, 3) 분포를 카테고리별로 집계하세요.
특정 번호에 정답이 과도하게 편중되어 있으면(전체의 40% 초과) 경고를 표시하세요.

## 출력 형식

### [카테고리명] (예: 성경)

| 번호 | 문제 ID | 문제 요약 (30자 이내) | 정답 번호 | 난이도 | 비고 |
|------|---------|----------------------|-----------|--------|------|
| N | bible_00N | ... | 0~3 | 상/중/하 | 난이도 부적합 시 권고 |

#### 정답 분포
- 0번: N개
- 1번: N개
- 2번: N개
- 3번: N개
- 편중 여부: 양호 / 경고 (N번에 편중)

---

마지막에 전체 요약을 출력합니다.

## 전체 요약
- 검토 범위: $1번 ~ $2번
- 총 검토 문제 수: N개
- 난이도 분포: 상 N개 / 중 N개 / 하 N개
- 난이도 부적합(하) 문제: N개 (있으면 ID 나열)
- 정답 분포 편중 카테고리: 없음 / [카테고리명]
