선생님 모드 통합 리포트를 생성합니다. 모든 분석을 한번에 실행하여 종합 보고서를 출력합니다.

---

## 실행 절차

아래 5개 분석을 순서대로 실행하여 하나의 통합 리포트로 출력합니다.
각 섹션 사이에 구분선(`---`)을 넣어 가독성을 확보합니다.

### 데이터 로드

아래 파일들을 모두 한번에 읽습니다:
- `quiz-game/teacher-data/results.json`
- `quiz-game/src/data/bible.ts`
- `quiz-game/src/data/hiphop.ts`
- `quiz-game/src/data/nba.ts`
- `quiz-game/src/data/mlb.ts`

파일이 없으면 `[오류] 필수 파일이 누락되었습니다: {파일명}` 출력 후 중단합니다.

---

### 출력 형식

```
# 선생님 모드 — 통합 리포트
생성일시: {YYYY-MM-DD HH:MM}
데이터: 학생 N명, 응시 기록 N건
```

---

### 섹션 1: 전체 성적 현황 (`/teacher-scores`에 해당)

학생별 카테고리 최고 점수 매트릭스를 출력합니다.

```
## 1. 전체 성적 현황

| 학생   | 성경 | 힙합 | NBA | MLB | 평균  | 응시 수 |
|--------|------|------|-----|-----|-------|---------|
| ...    |      |      |     |     |       |         |
```

---

### 섹션 2: 종합 순위표 (`/teacher-ranking`에 해당)

종합 순위와 카테고리별 TOP 3를 출력합니다.

```
## 2. 종합 순위표

### 종합 순위
| 순위 | 학생   | 합계   | 평균  |
|------|--------|--------|-------|
| 🥇   | ...    |        |       |

### 카테고리별 1위
| 카테고리 | 1위    | 점수 |
|----------|--------|------|
| ...      |        |      |
```

---

### 섹션 3: 문제 난이도 분석 (`/teacher-analysis`에 해당)

카테고리별 평균 정답률과 가장 어려운/쉬운 문제를 요약합니다.

```
## 3. 문제 난이도 분석

| 카테고리 | 평균 정답률 | 체감 난이도 | 가장 어려운 문제          | 가장 쉬운 문제          |
|----------|------------|-------------|--------------------------|------------------------|
| ...      |            |             |                          |                        |

### 주의 필요 문제 (정답률 40% 이하)
- {문제ID}: {문제 요약} — 정답률 N%
```

---

### 섹션 4: 학생별 취약 영역 요약 (`/teacher-weakness`에 해당)

```
## 4. 학생별 취약 영역

| 학생   | 가장 약한 카테고리 | 정답률 | 주요 오답 유형 | 타임아웃 |
|--------|--------------------|--------|----------------|----------|
| ...    |                    |        |                |          |

### 클래스 공통 취약 문제 TOP 3
1. {문제ID} — {문제 요약} (오답률 N%, N명 중 N명 오답)
2. ...
3. ...
```

---

### 섹션 5: 교사 액션 플랜

위 분석 결과를 종합하여 구체적인 교사 액션 항목을 제안합니다.

```
## 5. 교사 액션 플랜

### 즉시 조치 (우선순위 높음)
- [ ] {구체적 액션}: {사유}

### 보충 수업 제안
- [ ] {주제/카테고리}: {대상 학생} — {사유}

### 문제 품질 개선
- [ ] {문제ID}: {개선 내용}

### 다음 평가 시 참고
- {관찰 사항이나 추세}
```

---

### 마무리

```
---
리포트 생성 완료.
개별 상세 분석이 필요하면:
- `/teacher-scores {학생명}` — 특정 학생 상세 조회
- `/teacher-ranking {카테고리}` — 카테고리별 순위
- `/teacher-compare {학생1} {학생2}` — 학생 간 비교
- `/teacher-weakness {학생명}` — 개별 취약 영역 진단
```
