퀴즈 게임의 전체 통계를 분석하여 보고합니다.

`quiz-game/src/data/` 디렉터리의 bible.ts, hiphop.ts, nba.ts, mlb.ts 전체를 대상으로 합니다.

## 분석 항목

### 1. 카테고리별 기본 통계
각 카테고리 파일을 읽어 아래 정보를 집계합니다.
- 총 문제 수
- 정답 분포 (answer 값 0, 1, 2, 3 각각의 개수와 비율)
- 정답 편중 여부 (특정 번호가 전체의 40% 초과 시 경고)

### 2. 난이도 분석
각 문제를 아래 기준으로 난이도를 평가합니다.
- **상 (Hard)**: 구체적 수치, 통계, 연도 등 암기 없이는 풀 수 없는 문제
- **중 (Medium)**: 해당 분야에 관심 있는 사람이라면 추론 가능하지만 정확한 지식 필요
- **하 (Easy)**: 일반 상식 수준으로 풀 수 있는 문제

### 3. 출제 주제 분포
카테고리별로 어떤 주제/소재가 다뤄졌는지 분류합니다.
- bible: 구약/신약 비율, 인물/사건/수치 비율
- hiphop: 연대별 분포, 앨범/아티스트/레이블/사건 비율
- nba: 선수별 출제 빈도, 기록/통계/수상/시리즈 비율
- mlb: 선수별 출제 빈도, 기록/통계/수상/시리즈 비율

### 4. 품질 점검
- ID 형식 일관성: `{category}_{3자리 숫자}` 형식을 따르는지
- ID 연속성: 번호가 빠짐없이 연속인지
- 해설(`explanation`) 존재 여부: 빈 문자열이 없는지
- 선택지(`options`) 개수: 모든 문제가 정확히 4개인지
- 중복 문제: 같은 카테고리 내 유사한 주제의 문제가 있는지

### 카테고리별 상세

#### [카테고리명] (예: 📖 성경)
- **문제 수**: N개
- **정답 분포**:
  | 번호 | 개수 | 비율 | 상태 |
  |------|------|------|------|
  | 0 | N개 | N% | 양호/경고 |
  | 1 | N개 | N% | 양호/경고 |
  | 2 | N개 | N% | 양호/경고 |
  | 3 | N개 | N% | 양호/경고 |
- **난이도 분포**: 상 N개 / 중 N개 / 하 N개
- **주제 분포**: (주제별 개수)
- **품질 이슈**: 없음 / (이슈 나열)

---

### 종합 분석

#### 정답 분포 균형도
전체 문제의 정답 분포를 시각화합니다.
```
0번: ████████░░ N개 (N%)
1번: ██████░░░░ N개 (N%)
2번: █████████░ N개 (N%)
3번: ███░░░░░░░ N개 (N%)
```

#### 개선 권고사항
- (분석 결과에 기반한 구체적 권고 사항을 나열합니다)
- 예: "NBA 카테고리 정답 2번 편중 — 다음 문제 추가 시 0번 또는 3번 배정 권고"
- 예: "힙합 카테고리 2000년대 문제 부족 — 연대 다양성 확보 권고"
- 예: "성경 카테고리 난이도 '하' 문제 2개 — 난이도 상향 권고"
