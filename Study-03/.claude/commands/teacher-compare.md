두 명 이상의 학생 성적을 나란히 비교합니다.

인자: $ARGUMENTS (필수 — 비교할 학생 닉네임들을 공백으로 구분. 예: `민준 서연`)

---

## 실행 절차

### 0단계: 인자 검증

- 인자가 비어있으면 → `[오류] 비교할 학생 이름을 2명 이상 입력하세요. 예: /teacher-compare 민준 서연` 출력 후 중단
- 학생이 1명만 입력되면 → `[오류] 비교를 위해 최소 2명의 학생이 필요합니다.` 출력 후 중단
- 존재하지 않는 닉네임이 포함되면 → `[오류] '{닉네임}' 학생의 기록을 찾을 수 없습니다.` 출력 후 중단

### 1단계: 데이터 로드

`quiz-game/teacher-data/results.json` 파일을 읽고, 지정된 학생들의 데이터를 추출합니다.
동일 카테고리 복수 응시 시 최고 점수 기록을 사용합니다.

### 2단계: 나란히 비교표 출력

```
## 학생 비교: {학생1} vs {학생2} [vs {학생3} ...]

### 카테고리별 점수 비교
| 카테고리 | {학생1} | {학생2} | 차이  | 우위     |
|----------|---------|---------|-------|----------|
| 성경     | 8점     | 9점     | -1점  | {학생2}  |
| 힙합     | 7점     | 10점    | -3점  | {학생2}  |
| NBA      | 9점     | -       | -     | {학생1}  |
| MLB      | -       | 6점     | -     | {학생2}  |

### 종합 비교
| 항목           | {학생1} | {학생2} |
|----------------|---------|---------|
| 응시 카테고리  | N개     | N개     |
| 총점 (응시분)  | NN점    | NN점    |
| 평균 점수      | N.N점   | N.N점   |
| 평균 소요시간  | N초     | N초     |
| 최고 점수      | N점     | N점     |
| 최저 점수      | N점     | N점     |

### 강약점 비교
- {학생1}의 강점: {카테고리} (N점)
- {학생1}의 약점: {카테고리} (N점)
- {학생2}의 강점: {카테고리} (N점)
- {학생2}의 약점: {카테고리} (N점)

### 공통 응시 카테고리 심층 비교
(두 학생이 모두 응시한 카테고리에 대해서만)

| 문제ID    | 문제 요약       | {학생1} | {학생2} |
|-----------|----------------|---------|---------|
| bible_001 | 요셉이 형들에.. | ✅      | ✅      |
| bible_004 | 노아의 방주..   | ❌      | ✅      |

공통 오답 문제: {둘 다 틀린 문제 목록}
```

### 3단계: 교사 인사이트

```
### 교사 인사이트
- 전반적 우위: {학생명} (평균 N.N점 vs N.N점)
- 보완 추천:
  - {학생1}: {약한 카테고리} 영역 추가 학습 권장
  - {학생2}: {약한 카테고리} 영역 추가 학습 권장
- 공통 취약 문제가 있으면: 해당 문제의 해설 강화 또는 수업 보충 권고
```
