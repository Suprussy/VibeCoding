퀴즈 데이터 파일에서 최상급 표현이 포함된 문제를 검사합니다.

$ARGUMENTS 값이 있으면 해당 카테고리(bible, hiphop, nba, mlb)의 파일만 검증하고,
값이 없으면 `quiz-game/src/data/` 디렉터리의 bible.ts, hiphop.ts, nba.ts, mlb.ts 전체를 검증합니다.

각 Question의 `question` 필드에서 아래 표현이 포함된 항목을 찾아 목록으로 출력하세요.

검사 대상 표현: 가장, 최초, 최대, 최고, 최소, 유일, 처음, 단 하나, 역대, 사상 첫, 사상 최

이러한 표현이 감지되면 측정 기준(범위, 시점, 리그/분류 등)이 문제 텍스트에 명시되어 있는지 확인하고,
기준이 없으면 어떤 기준을 추가해야 하는지 구체적인 수정안을 제시하세요.

출력 형식:
## 최상급 표현 검증 결과

각 파일별로 섹션을 나누어 아래 형식으로 출력합니다.

### [파일명] (예: bible.ts)
- **[문제 id]** `[감지된 표현]` — [question 전체 텍스트]
  - 기준 명시 여부: O / X
  - 권고: [기준이 없으면 수정 제안, 있으면 "양호"]

최상급 표현이 하나도 없는 파일은 "이상 없음"으로 표시합니다.

마지막에 전체 요약을 출력합니다.
## 요약
- 총 검사 문제 수: N개
- 최상급 표현 감지: N개
- 기준 미명시(수정 권고): N개
