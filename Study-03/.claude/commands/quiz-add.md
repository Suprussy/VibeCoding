$1 카테고리에 난이도 $2의 새 퀴즈 문제 1개를 추가합니다.

- $1: 카테고리 (bible, hiphop, nba, mlb)
- $2: 난이도 (상, 중, 하)

## 진행 절차

### 1단계: 대상 파일 확인
`quiz-game/src/data/$1.ts` 파일을 읽어서 현재 문제 수와 마지막 id 번호를 확인합니다.
새 문제의 id는 `$1_0XX` 형식으로, 마지막 번호 + 1로 설정합니다.
예: 현재 마지막이 `bible_010`이면 → `bible_011`

### 2단계: 기존 정답 분포 확인
해당 파일의 모든 문제에서 `answer` 값(0, 1, 2, 3)의 분포를 집계합니다.
가장 적게 사용된 번호를 새 문제의 정답 번호로 우선 배정합니다.
(동률이면 0 > 3 > 1 > 2 순서로 우선)

### 3단계: 문제 작성

난이도 기준:
- **상 (Hard)**: 구체적 수치, 통계, 연도 등 암기 없이는 풀 수 없는 문제. 정답과 오답 선택지 간 구분이 어려움.
- **중 (Medium)**: 해당 분야에 관심 있는 사람이라면 추론 가능하지만, 정확한 지식이 필요한 문제.
- **하 (Easy)**: 일반 상식 수준. 단, 마니아 퀴즈 취지상 가급적 사용하지 않습니다.

카테고리별 출제 방향:
- bible: 구약/신약 균형, 구체적 수치·인물·사건 (예: 팔린 가격, 재위 기간 등)
- hiphop: 90~2010년대 중심, 앨범명·피처링·레이블·역사적 사건
- nba: 역대 기록·통계·MVP·파이널 수치 기반
- mlb: 역대 기록·월드시리즈·전설적 선수 수치 기반

### 4단계: 검증 가이드라인 (반드시 준수)

아래 항목을 모두 확인한 후에만 문제를 추가합니다.

1. **정답 유일성**: 정답이 하나뿐인가? 다른 해석이 가능하면 조건을 명시합니다.
2. **최상급 표현 기준**: '가장 큰', '최초의', '역대' 등의 표현에 측정 기준(범위, 시점, 리그/분류)이 명시되어 있는가?
3. **시간·범위 명확성**: 변할 수 있는 정보에는 시점을 명시합니다. 지리적·분류적 범위를 한정합니다.
4. **교차 검증**: 정답 사실이 정확한지 확인합니다. 의심스러운 정보는 2개 이상 출처 기준으로 판단합니다.
5. **기존 문제와 중복 없음**: 같은 파일 내 유사한 주제의 문제가 이미 있는지 확인합니다.
6. **오답 선택지 그럴듯함**: 오답 4개가 너무 명백하게 틀리지 않아야 합니다. 실제로 헷갈릴 수 있는 값으로 구성합니다.
